keyboard 快捷键参考

  • Ctrl + Enter (或 Cmd + Enter) - 保存笔记
  • Ctrl + E (或 Cmd + E) - 导出数据
  • Ctrl + I (或 Cmd + I) - 导入数据
  • Esc - 取消编辑 / 清除搜索 / 关闭弹窗
  • / - 快速聚焦到搜索框
  • 双击笔记卡片 - 进入编辑模式
  • 点击标签 - 快速搜索该标签

📁 自动备份功能说明

  • 静默备份: 每天首次保存时会提示，后续保存静默自动备份
  • 文件格式: flomo-auto-backup-YYYY-MM-DD_HH-MM-SS.json
  • 保存位置: 浏览器的默认下载文件夹
  • 备份限制: 每天最多10次，每5分钟最多1次
  • 数据保护: 清理浏览器前，请确保已保存备份文件
